# Midillon
Carillon MIDI permettant de jouer un carillon virtuellement depuis un clavier MIDI ou un fichier MIDI, et de générer un fichier de programmation de jeu automatique de carillon à partir d'un fichier MIDI.
Midillon peut utiliser les échantillons sonores d'un vrai carillon (un fichier .wav par cloche, nommé avec le nom de la note de la cloche).
Un fichier de définition de carillon peut être fourni à Midillon pour simuler un clavier, un pédalier et un jeu automatique d'un vrai carillon (voir l'exemple fourni).
